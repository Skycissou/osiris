Recettes guidées

Des parcours en langage humain. L’utilisateur choisit son intention, l’interface construit les appels API et le rapport.

Vérifier une entreprise

SIREN/SIRET, statut, siège, établissements, BODACC, export.

1Entrer nom ou SIREN
2Comparer les correspondances
3Générer une fiche sourcée

Explorer un territoire dans le temps

Adresse ou commune → carte IGN → remonter aux orthophotos anciennes → surcouches cadastre / agricole.

1Géocoder & basculer en vue Carte
2Remonter le temps (ortho par année, photos N&B, État-Major)
3Empiler cadastre, RPG, hydro…

Créer un rapport OSINT admin

Un rapport propre, avec sources officielles, limites et zones non vérifiées.

1Sélectionner les sources
2Classer la preuve
3Exporter PDF/Markdown

Surveiller une zone en direct

Cockpit carte : vols, séismes, feux, géopolitique et cyber en temps réel sur une emprise choisie.

1Ouvrir « Cockpit carte »
2Activer les couches + filtres
3Preset de vue + partage du lien

Investiguer une IP / un domaine

Boîte OSINT : détection auto du type de cible → whois, DNS, géoloc IP, certificats, CVE, réputation.

1Coller la cible (IP, domaine, email…)
2Lancer les lookups en parallèle
3Étendre en graphe d'entités

Tracer un vol ou un VIP

Couche vols ADS-B : suivre un aéronef, voir sa traînée, sa fiche (photo, cap, altitude) et l'alerte VIP.

1Activer la couche « Vols »
2Cliquer l'aéronef → fiche
3Filtrer (militaire, VIP, altitude)

Glossaire

Tous les sigles de l'app, expliqués simplement. Survole aussi les termes soulignés (pointillés) partout dans OSIRIS pour la définition à la volée.

Open data FR (recherche & entreprises)
SIREN / SIRET
Identifiants d'entreprise (SIREN = 9 chiffres, l'entité) et d'établissement (SIRET = SIREN + 5 chiffres, le lieu).
BODACC
Bulletin Officiel des Annonces Civiles et Commerciales : ventes, procédures collectives, radiations.
RNA
Répertoire National des Associations : identifiant et déclarations des associations (loi 1901).
DVF
Demandes de Valeurs Foncières : base publique des ventes immobilières (prix, date, surface).
NAF / APE
Nomenclature d'Activités Française : code du secteur d'activité d'une entreprise (ex. 10.71C).
INSEE
Institut national de la statistique : produit les codes communes/pays et de nombreux référentiels.
BAN
Base Adresse Nationale : référentiel officiel des adresses françaises (géocodage).
GTFS
General Transit Feed Specification : format standard des données de transport (arrêts, lignes, horaires).
IGN / Géoplateforme
Institut national de l'information géographique : fournit les fonds de carte, orthophotos et surcouches (WMTS gratuit).
WMTS
Web Map Tile Service : standard de diffusion de cartes en tuiles (utilisé pour les fonds IGN).
Cockpit carte & temps réel
ADS-B
Automatic Dependent Surveillance–Broadcast : position des avions diffusée en clair, captable par des récepteurs publics.
AIS
Automatic Identification System : équivalent maritime, signal de position des navires.
TLE / SGP4
Two-Line Element : paramètres orbitaux d'un satellite, propagés par le modèle SGP4 pour le situer en direct.
USGS
United States Geological Survey : agence géologique américaine, flux public des séismes.
FIRMS / VIIRS
Système NASA de foyers thermiques actifs (feux) ; VIIRS est le capteur satellite utilisé.
GDELT
Global Database of Events, Language and Tone : événements mondiaux géolocalisés issus de la presse.
C2
Command & Control : serveur pilotant un logiciel malveillant. Suivi ici en veille défensive (abuse.ch).
OSINT & cyber
OSINT
Open Source Intelligence : renseignement à partir de sources ouvertes/publiques, cadre légal et défensif.
WHOIS / RDAP
Données d'enregistrement d'un domaine ou d'une IP (RDAP = version moderne du WHOIS).
DNS
Domain Name System : annuaire qui traduit un nom de domaine en adresse IP.
IP / ASN / BGP
Adresse réseau d'une machine · numéro d'un opérateur réseau · protocole de routage entre réseaux.
CVE
Common Vulnerabilities and Exposures : identifiant public d'une faille de sécurité connue.
Shodan · HIBP · AbuseIPDB
Moteur d'exposition réseau · fuites liées à un email (Have I Been Pwned) · réputation d'une IP.
Cadre & technique
ARPD
Cadre associatif de police auquel Cissou est rattaché comme référent OSINT (Occitanie).
RGPD
Règlement Général sur la Protection des Données : cadre légal européen sur les données personnelles.
API
Application Programming Interface : porte d'entrée logicielle pour interroger une source de données.
Forme 1 / Forme 2
Forme 1 = tout public (ARPD, défaut). Forme 2 = enquêteur : débloque les couches sensibles, avec consentement.
MapLibre
Moteur cartographique open-source (rendu de la vue Carte et du cockpit).

Garde-fous intégrés

Le produit doit être utile sans devenir intrusif. La confiance vient autant des limites que des résultats.

Sources visibles

Chaque résultat indique : officiel, secondaire, dataset local, bloqué ou habilitation nécessaire.

Personnes physiques

Masquage par défaut des adresses personnelles et pas de scoring opaque sur individus.

Rapports propres

Export avec requêtes utilisées, dates, limites et “ce qu’on ne peut pas conclure”.

Carte : agréger, pas surveiller

La carte ne fait qu’afficher des données déjà publiques — fonds & imagerie officiels IGN (Géoplateforme), sièges d’entreprises du registre. Lecture seule, aucune donnée personnelle ajoutée, sources et millésimes visibles.
